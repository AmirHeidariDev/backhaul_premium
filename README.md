# backhaul_premium

آپدیت سرور: 
```bash
apt update && apt upgrade -y

```

نصب zip : 
```bash
apt install unzip

```

برای نصب بکهال : 
```bash
sudo wget https://raw.githubusercontent.com/AmirHeidariDev/backhaul_premium/main/backhaul.zip
```

خارج کردن از zip :
```bash
apt install unzip

```

ران کردن برنامه : 
```bash
chmod +x /root/backhaul-core/
chmod +x /root/backhaul-core/backhaul_premium
chmod +x /root/backhaul.sh

bash /root/backhaul.sh
```

اجرا کردن مجدد تانل :

```bash
bash /root/backhaul.sh
```
